
# Simple Laravel 5.4 CRUD application
## Installation
```
$ composer install && npm install
```

Open ```.env``` and enter necessary config for DB and Oauth Providers Settings.

```
$ php artisan migrate
$ php artisan db:seed
```

## Work Flow

**General Workflow**

Open terminal
```
$ php artisan serve
```
Open new terminal
```
$ npm run dev
```
